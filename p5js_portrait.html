P5.js Self Portrait

04/11/2026

For this assignment, we were tasked with using p5.js, a programming library for creating visual art through code, to construct a self-portrait using geometric shapes. I chose to render my portrait in red and black, as these are my favorite colors and align with the color scheme of this portfolio website.
